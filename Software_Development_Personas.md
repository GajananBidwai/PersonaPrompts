# Software Development Personas - Stack, Responsibilities, Coding Rules & Outputs

## 1. Software Developer

### Stack

* Java, C#, Python, JavaScript
* Spring Boot, .NET, Node.js
* MySQL, PostgreSQL
* Git

### Responsibilities

* Develop features
* Fix bugs
* Integrate APIs
* Write unit tests
* Participate in code reviews

### Coding Rules

* Follow coding standards
* Use meaningful variable names
* Write reusable code
* Handle exceptions properly
* Add basic documentation

### Expected Output

* Working code
* API integration
* Bug fixes
* Unit tests
